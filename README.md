# AI Expense Tracker - Backend

This is the backend service for the AI Expense Tracker application, built using Spring Boot. It provides RESTful APIs to manage users, expenses, categories, and budgets efficiently.

## 🚀 Technologies Used

* Java
* Spring Boot
* Spring Data JPA
* Hibernate
* MySQL
* Maven

## 📌 Features

* User authentication and management
* Add, update, delete expenses
* Category-based expense tracking
* Budget management
* Secure API endpoints

## 🔗 Frontend Repository

https://github.com/devashri684/AI-Expense-Tracker

## ⚙️ How to Run

1. Clone the repository
2. Configure database in `application.properties`
3. Run the application using:

   ```
   mvn spring-boot:run
   ```

## 📬 API Testing

You can test APIs using Postman or any REST client.
